    Bandeau de texte défilant

    Bannière publicitaire (images)

    Jusqu'à 5 bannières, affichées en rotation automatique toutes les 7 secondes sur la page d'accueil. Taille standard recommandée : 1600 × 500 px (ratio 16:5) — les images sont automatiquement recadrées à ce format pour un rendu homogène.

    Limite de 5 bannières atteinte — supprimez-en une pour en ajouter une nouvelle.

    Bannière latérale (verticale)

    Affichée à côté du catalogue, sur toute sa hauteur. Format idéal : 300 × 600 px (portrait). Sur mobile, elle s'affiche sous le catalogue dans un format plus large et moins haut. Laisse vide pour la masquer.

    Bannières carrées (sous la bannière latérale)

    Deux emplacements, affichés côte à côte sur mobile. Format idéal : 300 × 300 px (carré). Laisse vide pour masquer un emplacement.

    Pages légales

    Ces textes sont accessibles depuis le bas de page du site (footer).

    Laissez une ligne vide entre deux paragraphes pour les séparer sur la page publique.
